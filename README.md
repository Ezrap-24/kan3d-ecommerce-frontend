# kan3d-ecommerce-frontend
Frontend de tienda e-commerce Kan3D Store de una tienda de e-commerce ficticia llamada **Kan3D Store**, dedicada a la venta de productos impresos en 3D (soportes para controles, figuras y accesorios personalizados).

## Objetivo del proyecto

- Practicar los fundamentos del desarrollo web.
- Construir la interfaz de una tienda online con navegación.
- Servir como parte de mi portafolio en GitHub.

## Tecnologías utilizadas

- **HTML5**: estructura de las páginas.
- **CSS3**: estilos personalizados.
- **Bootstrap 5 (CDN)**: sistema de grillas y componentes responsivos.
- **JavaScript**: archivo `assets/js/main.js` para interactividad básica.

## Estructura del sitio

- `index.html`: página principal con:
  - Barra de navegación.
  - Listado de productos en tarjetas.
  - Botones de "Comprar" y "Ver detalles".
  - Link a la página de carro de compras.
  - Hero con imagen real de la impresora.
  - Footer con información de la tienda.

- `producto.html`: página de descripción del producto real:
  - Soporte impreso en 3D para control de PS5.
  - Imágenes del producto.
  - Detalle de características y precio.
  - Botón "Agregar al carrito" y enlace para volver al catálogo.

- `carrito.html`: página simple con el texto "Carro de compras en construcción", utilizada como destino para el enlace del menú.

Este repositorio representa la **parte de frontend** del proyecto de e-commerce solicitado en el Módulo 2.
